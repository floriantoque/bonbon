# Story Creator App

A Gradio web application for creating and saving interactive stories.

## Features

- Generate stories with customizable parameters:
  - Number of phrases
  - Characters
  - Target audience age
  - Language (French or English)
  - Story context
  - Number of breakpoints
  - Breakpoint symbol
- Save generated stories to files

## Installation

```bash
uv sync --reinstall
```

## Usage

```bash
uv run app.py
```
